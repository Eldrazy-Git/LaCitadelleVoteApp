Journal des versions

Historique des mises à jour de l’application La Citadelle Vote

Dernière version : 1.1_3 (Stable) — novembre 2025

Version 1.1_3 — Release Stable

  • Correction des notifications manquantes lorsque l’application est fermée ou l’appareil verrouillé.
  • Cliquer sur un bouton de vote ne redémarre plus le timer (évite les erreurs de missclicks).
  • Ajout d’une bulle d’information indiquant le temps restant avant le prochain vote.
  • Cliquer sur la notification relance le timer du site et ouvre directement la page de vote.
  • Cliquer sur un groupe de notifications n’ouvre plus qu’une seule instance de l’application.

Bugs connus : notifications fantômes rares, fond d’icônes dégradé sur certaines versions.

Version 1.1_2

  • Ajout d’un nouveau site de vote : serveur-minecraft-vote.fr.
  • Positionné sous serveur-prive.net (même durée de 90 minutes).

Bugs connus : notifications fantômes occasionnelles et fond d’icônes déformé.

Version 1.1

  • Refonte complète pour compatibilité Android 10 à 15.
  • Nouvelle architecture des tâches de fond et des notifications (WorkManager).
  • Cliquer sur un bouton de vote réinitialise le timer sans duplication.
  • Notifications fiabilisées, son custom réparé.
  • Interface modernisée, suppression du bouton de paramètres visible.
  • Cliquer sur la notification ouvre directement l’application de vote.

Bugs connus : notifications absentes sur certains appareils (Xiaomi notamment).

Versions 1.0 à 1.0_4 — Versions de test initiales

  • Version initiale de l’application avec les 3 sites de vote intégrés :
    • serveur-prive.net
    • serveur-minecraft.com
    • serveursminecraft.org
  • Ajout du logo cliquable vers le site officiel et du logo Discord.
  • Ajout des timers par site et notifications personnalisées “Vote disponible”.
  • Utilisation de la police médiévale MedievalSharp.ttf.
  • Bouton paramètres : réinitialisation des compteurs, test des notifications.

Bugs connus : duplication des timers, son de notification parfois absent, fond d’icônes imparfait.

Ce journal est mis à jour automatiquement à chaque release publique.
